From this video, https://www.youtube.com/watch?v=B0g7sxoYfxk

Settings > Repository > Expand Mirroring repositories

Past the SSH link from the Github repo 
    git@github.com:PaulEdworthy/repo.git
Edit the link
    ssh://git@github.com/PaulEdworthy/test-repo.git
    Mirror direction: Push (to Github) or Pull (from Github)
    Press Detect Host Keys
Change the validation from Password to SSH public key
Press Mirror Respository

Once it's mirrored, copy the clipboard contents to copy your host Keys
Go to the Github repo
Settings > Deploy Keys to create a new one
    Paste in the keys
    Check to allow write access (or it won't work)

Go back to Gitlab and refresh the mirror.
    Double check that it worked on Github also.
